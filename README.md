# Dado Eletrônico com Arduino

Projeto de um dado eletrônico usando Arduino no Tinkercad.

## Componentes
- Arduino Uno
- 6 LEDs
- 6 resistores 220Ω
- 1 botão
- 1 resistor 10kΩ (para o botão, se não usar INPUT_PULLUP)

## Funcionamento
Ao apertar o botão, um número aleatório de 1 a 6 é gerado e os LEDs correspondentes acendem.
